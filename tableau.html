Flu Shot Report Dashboard

Project Overview

This Tableau dashboard provides an interactive visualization of flu shot distribution trends across different demographics. The goal of this project was to analyze flu shot coverage and identify trends based on factors like age, location, and time.

Key Insights

  • Trends in flu shot adoption across various demographics.
  • Comparison of flu shot coverage by year and region.
  • Identified key areas where flu shot distribution could be improved.
